# eval_off_site_construction
This capstone created a cost estimation and facility location model that aims to quantify the cost reduction opportunities and evaluate labor and transportation tradeoffs of off-site housing construction. 

This work was completed as part of an [MIT Supply Chain Management](https://scm.mit.edu/) Capstone Project (May, 2026): Evaluating Cost and Supply Chain Trade-offs in Off-site Construction

---
# Cost Estimation Model:
This model was developed to approximate the cost savings of multiple off-site configurations compared to traditional, site-built construction. Users can view the definitions of each off-site configuration as well as modify the parameters for more detailed scenarios. 

# Facility Location Model:
This model was designed to evaluate the specific tradeoff between labor and transportation costs. Model results show the selected central facility location to service the housing demand based on these two variables.
